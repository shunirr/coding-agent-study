# Part 3: 会話履歴付きチャット

Part 2 をベースに、REPL ループで会話を続けられるようにします。会話履歴を蓄積することで、文脈を維持した対話が可能になります。

## ポイント

- `messages` 配列に `user` / `assistant` メッセージを蓄積して文脈を維持する仕組み
- REPL ループによる対話型インターフェースの実装
- 会話履歴がトークン消費に与える影響（コメントで注記）

## ファイル構成

- `main.ts` — メインスクリプト（チャットループ付き）
- `SYSTEM_PROMPT.md` — 会話の継続性を意識したシステムプロンプト

## 実行

```bash
deno run --allow-net --allow-read main.ts
```

終了するには `Ctrl+C` を押してください。

## Part 2 からの変更点

- `while (true)` による REPL ループを追加
- LLM のレスポンスを `messages` に追加して会話履歴を蓄積
